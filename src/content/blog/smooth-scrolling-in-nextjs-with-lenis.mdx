## Using Lenis in a Next.js 16 App

> üí° Tip: Even though Lenis is lightweight, always test on a new branch before pushing to production.

### Install Lenis

Using your preferred package manager:

```bash
npm install lenis
# or
yarn add lenis
# or
pnpm add lenis
```

## Add `<ReactLenis>` to your layout

Head over to your `layout.tsx` (root layout or a custom layout component ‚Äî recommended).

Import `ReactLenis` component from `lenis/react`:

```tsx
import { ReactLenis } from "lenis/react";
```

Call `ReactLenis` before the components:

```tsx
return (
  <>
    <ReactLenis root />
    {/* content */}
  </>
);
```

For example:

```tsx
import { ReactLenis } from "lenis/react";
import { Header } from "@/components/header";
import { Footer } from "@/components/footer";

export default function Layout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <>
      <ReactLenis root />
      <Header />
      {children}
      <Footer />
    </>
  );
}
```

That‚Äôs it. You now have buttery smooth scrolling in your Next.js app üßà

## Customizing the Scroll Behavior

You can fine-tune how Lenis behaves by passing options to the `<ReactLenis>` component:

```tsx
<ReactLenis root options={{ smoothWheel: true, lerp: 0.1 }} />
```

Here‚Äôs what some options mean:

- `smoothWheel`: enables smooth wheel scrolling
- `lerp`: controls scroll interpolation (lower = smoother but slower)
